# **App Name**: CodeCalc

## Core Features:

- Code Generation: Generate Java and XML code snippets for scientific calculator functionalities using an AI tool.
- Code Display: Display the generated code in separate, well-formatted panels for Java and XML.
- Copy to Clipboard: Provide options to copy the generated code snippets to the clipboard.

## Style Guidelines:

- Primary color: Dark blue (#283593) for a professional look.
- Secondary color: Light gray (#f5f5f5) for background.
- Accent: Teal (#009688) for buttons and highlights.
- Sans-serif fonts for readability.
- Use clear and simple icons for buttons and functions.
- Split the screen into two main panels, one for Java code and one for XML code.
